# Rule Loading Guide

When working on this codebase, load the appropriate rule files based on what you're editing:

- **Backend Python files** (`app/**/*.py`): Load `backend-architecture.mdc`
- **Frontend React components** (`frontend/src/**/*.tsx`): Load `frontend-react-patterns.mdc`
- **Frontend TypeScript files** (`frontend/src/**/*.{ts,tsx}`): Load `react-query-and-client.mdc`

If working across both backend and frontend, load all relevant rules.
